Cloud Firebase Setup Instructions

Connect this applet directly to a live Google Cloud Firebase Project by following these quick steps:

  1. Go to the Firebase Console and click Add Project.
  2. Once created, select the web platform icon </> to register a new Web App.
  3. Copy your unique firebaseConfig credentials from the setup screen.
  4. Open this file (index.html) in your code editor and look for the firebaseConfig constant variable near the very top of the script tag (around line 520).
  5. Replace the default configuration values with your actual Firebase API keys:
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "your-project-id.firebaseapp.com",
  projectId: "your-project-id",
  storageBucket: "your-project-id.appspot.com",
  messagingSenderId: "your-sender-id",
  appId: "your-app-id"
};

Deploying to GitHub Pages:

Simply push this single, fully self-contained index.html file to a public GitHub repository. Navigate to your Repository Settings > Pages, and enable deployment from your main branch. Your site will instantly go live around the world for free!
